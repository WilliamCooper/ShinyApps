Guide to this Presentation:



The main tabs, after “Guide”, are:

  1. Background: What is a Shiny app? How are they supported in EOL? How are they being used now? What are potential uses?

  2. Instructional and tutorial examples: SEGUE examples; R tutorial; tutorial re variance spectra (often called power spectra), etc.

  3. Data Perusal and QA: Two examples of more elaborate Shiny apps are included under this tab, a tool for perusal of RAF data files and a quality-assurance tool being used to monitor data collections in RAF projects during field projects. These two applications have many options, so they will only be presented here in cursory fashion, but both have extensive manuals that provide complete information regarding usage.

  4. Special Processing: Additional processing tools are available that apply a Kalman filter to wind measurements, add new processing options for winds, and add a surface-elevation at the aircraft location to existing data archives. These will not be demonstrated and will only be mentioned as additional capabilities available using this “Demo” Shiny app.
