Checking Temperature: Recommendations

  1. Start with the "Review" tab; see plots 3 and 4. Use "brush" to zoom to problems. Look for general consistency with past projects.

  2. Use the "Past Projects" tab and the secondary tab "ATX". See all three comparisons:

    • comparison to prediction based on AT_A (which uses the expected relationship from past projects).

    • comparison among ATs: select pairs or sets to compare, see comparison vs altitude, expect consistency with past projects.

    • comparison to a prediction from integrating the hydrostatic equation; expect agreement to within a few degrees.

  1. In "Known Problems" -> "in-cloud check" look for consistency between the dewpoint and the temperature when in cloud. This is not useful except when the VCSEL provides the reference for DPXC and when its calibration is good.
