Checking INS measurements: Recommendations

  1. Start with the "Review" tab; see plots 18 and 19, which show how the backup measurements from the second INS compare to the standard measurements. Use "brush" to zoom to problems. Look for general consistency with past projects.

  2. Use the "Past Projects" -> "INS/IRU comparisons" tab to see this same comparison in the form of histograms of the differences.

  3. Also under "Past Projects" -> "PSXC", see the "hydrostatic-eq check" tab; measurements should be consistent to within a few hPa.
