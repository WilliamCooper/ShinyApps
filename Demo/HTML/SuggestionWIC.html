Checking the Vertical Wind: Recommendations

  1. Start with the "Review" tab, plot 13, which shows (at the bottom) the vertical wind. Use "brush" to zoom to problems. Look for general consistency with past projects and a reasonable mean wind. Plot 17 shows the comparison between the measured AKRD and the prediction for zero wind (AOAREF); these should be in reasonable agreement except in cases with significant vertical wind.

  2. Use the "Past Projects" -> "Maneuvers" --> "Speed Run" and ... -> "Pitch" tabs if there are any corresponding maneuvers available. There is guidance under these tabs regarding evaluating these maneuvers, which can also be compared to maneuvers from earlier projects.

  3. Also under "Past Projects" -> "PSXC", see the "hydrostatic-eq check" tab; measurements should be consistent to within a few hPa.
